Configure Two-factor Authentication

Steps

  1. Install an authenticator app like Google Authenticator on your phone.
  2. Scan the QR code below with your phone camera.
  3. Tap "Open in Authenticator".
  4. When asked "Do you want to add the token ...", tap "Yes".
  5. Enter the code displayed in the authenticator app in the input below.
  6. Press the "Submit" button.

After following these steps, you will be asked to supply a code from your authenticator app each time you login.
